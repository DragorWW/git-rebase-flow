# Git rebase flow

## ⛳ Benefits
 - simple work with dependent tasks
 - pretty history

## 🚦convention
- ❌ Merge commit
- ❌ Development process history
- ❌ commit without task
- 1️⃣ Task – 1️⃣ commit
- 1️⃣ Branch – 1️⃣ developer
- 1️⃣ Main branch
- ✅ Rebase
- ✅ Stateless release
- ✅ Rebase only task branches
- ✅ Atomic tasks

## 🛠 Tools
- ⚙️  Automatization
- 🏗 Decomposition
- 🚩 Feature flags
- 🔍 Testable changes
- 🔗 Backward compatibility
